4. Javascript Scope

We have two type of Scope :

Scope determines the accessibility / visibility of variables.

  1. Global
    • Global means outside a function or block
    • Declaring variable outside a function or a block
    • We can access global scope anywhere in code
    • If we assign a value with a variable without declared that variable is automatically beacame global scope.
  2. Local
    • Local means inside a function or a block
    • We can only access locally or inside a block
    • We can not access local scope from global
  3. Functional
    • Same as local scope
